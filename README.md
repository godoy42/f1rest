
Exemplo de API
=========
API simples para listagem de dados da F1
Feito em Python com Flask

Uso
---------------

Executar com [Docker](https://www.docker.com/products/docker-desktop).

    docker build -t f1 .
    docker run -d -p 5000:80 f1

Pode também ser executado localmente.

    pip install -r requirements.txt
    python f1.py
